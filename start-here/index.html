What This Is (and What It Is Not)

What this is:

A structured, continuously-updated ecosystem of:

  • Clinical guidance that helps you interpret symptoms, risk, and next steps
  • Operating intelligence that explains what has changed in law/policy and how it affects care
  • Tools + scripts designed for real-world constraints (time, fear, dismissal, bureaucracy)
  • Community built around clarity, accountability, and protection

What this is not:

  • Not a substitute for emergency care or ongoing medical care
  • Not a "wellness lifestyle" publication
  • Not medical advice for your individual case

The Infrastructure: Packs + Playbooks

We are transitioning from a passive library to an active infrastructure.

Navigation Guide
Playbooks (Action Pathways)

A Playbook is a sequenced pathway for a specific situation (example: recurrent vaginitis in pregnancy, hyperemesis, endometrial biopsy). It tells you what to do in order, what matters, what's noise, and what to ask for.

Packs (System Modules)

A Pack is a higher-level bundle of related playbooks (example: Pregnancy Symptoms That Don't Behave). Packs are how you navigate a bigger domain without getting lost.

Translation

If you're overwhelmed, start with a Pack. If you're in a specific problem, start with a Playbook.

Quick Safety Note

If you think you might be having an emergency — heavy bleeding, severe pain, trouble breathing, fainting, seizures, severe headache + vision changes, chest pain, signs of preterm labor, or "something feels dangerously wrong" — seek urgent care now or call emergency services. Labora helps you prepare, interpret, and advocate. It does not replace emergency evaluation.

Where to Begin?

Start with the path that matches your reality:

  • If symptoms don't match tests → start with a Playbook
  • If you're pregnant and something feels off repeatedly → start with a Pregnancy Pack
  • If you're a birthworker supporting clients under chaos → start with Operating Intelligence + birthworker-facing pathways
  • If you're here to understand the policy mechanics behind clinical harm → start with In Her Name
